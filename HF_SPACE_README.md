# Diabetes Risk Predictor — End-to-End MLOps

A screening-triage tool that estimates a patient's probability of having type-2 diabetes from routine primary-care measurements. Built with the Zimbabwean primary-care setting in mind — where HbA1c testing isn't universally available and a fast triage can route the right patients to the right test.

**Source code:** https://github.com/nanettetada/diabetes-risk-mlops

**Stack:** Python · scikit-learn · MLflow · Plotly · Streamlit · FastAPI · Docker · GitHub Actions

---

This Space is a deployment of the dashboard. For the full README with badges, screenshots, methodology, and the model card, see the **[GitHub repo](https://github.com/nanettetada/diabetes-risk-mlops)**.

**This is not a diagnostic device.** It's a screening probability. Any real-world use would require regulatory approval and prospective validation on the local population.
